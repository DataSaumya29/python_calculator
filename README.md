# python_calculator

A simple command-line calculator built using Python.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Continuous usage (loop)

## Concepts Used
- Functions
- Loops
- Conditional statements
- User input handling

## How to Run

1. Open terminal
2. Run:
   python calculator_functions.py
